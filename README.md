# Music Streaming App: User Engagement Dashboard

## 📌 Project Overview
An end-to-end data analytics project mapping out user habits for a music streaming service. This project cleans raw mock data, utilizes multi-layered Pivot Tables, implements KPI metrics via cross-sheet live formulas, and presents executive business recommendations.

## 🛠️ Tools Used
* **Microsoft Excel:** Data Cleaning, Formulas (COUNTA, AVERAGE), Pivot Tables, Custom Formatting, Dashboard Design.

## 📈 Key Metrics
* **Total Active Sample Users:** 15
* **Average Platform Retention:** 57.7 Hours/Month per User
* **Peak Engagement Streams:** Hip-Hop & Pop Genres (58 Hours/Month)

## 📁 Repository Contents
* `MUSIC STREAMING ENGAGEMENT ANALYSIS.xlsx`: Completed workbook containing the raw data sheet, discrete analysis tabs, and the finalized interactive executive dashboard.
